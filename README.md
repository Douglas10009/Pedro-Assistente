# Pedro-Assistente
Pedro, o seu assistente pessoal



## Ele vai te ajudar com:

- Dizendo as horas
- Procuras pela internet
- Tocar músicas
- Abrir o navegador (Chromme)

## Dependências
- pip install SpeechRecognition #Reconhecimento de voz
- pip install pyttsx3 #Transformar texto em áudio
- pip install PyAudio
